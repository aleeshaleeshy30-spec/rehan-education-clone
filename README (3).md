# Rehan Education Clone

A modern, fully functional web clone of **Rehan Education** built for **CWI TASK : 45**.

## ✨ Features

- **Homepage**: Hero banner, mission, statistics, and program introduction
- **Curriculum Page**: Detailed view of Digital Skills, AI, and Financial Literacy programs
- **Facilitators Page**: Team member profiles
- **Contact Page**: Contact details + fully working contact form
- Smooth single-page navigation
- Responsive design for mobile & desktop
- Professional, modern educational UI

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- Vanilla JavaScript
- Font Awesome
